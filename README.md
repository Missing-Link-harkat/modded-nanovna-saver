## Run Guide

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Create Virtual Environment

```bash
python3 -m venv venv
```

or

```bash
python -m venv venv
```

### Open Modded NanoVNA-Saver

```bash
cd ~/nanovna-saver
source venv/bin/activate
export PYTHONPATH=src
python -m NanoVNASaver
```
